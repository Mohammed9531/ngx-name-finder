How to play name finder?

  • Step-1: Please think of a name of person. It can be either a first name or the last name. Please note that the name finder is entirely based on alphabets/characters, no special characters would be guessed.
  • Step-2: Alright, now don't reveal the name to anyone, keep it to yourself.
  • Step-3: In the below given alphabetical grid, please select the line that holds the alphabet in sequence. For instance, say 'Harry' be the name you chose. Select the line that contains the letter 'H' then repeat this step for all the subsequent letters 'A, R, R, Y'.
  • Step-5: When finish, we will guess the name of the person.
